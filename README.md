## Selected AI Work

**Knowledge Graph Based RAG on Customer AI**  
Built a GraphRAG system with LightRAG and LanceDB for complex entity-relationship retrieval over multimodal text/image data. Added embedding fallback across Qwen3, BGE, and MiniLM, plus LLM caching and sentiment-based escalation routing.

**S-TEA: Sequential Slow Transfer Learning**  
Designed a continual learning framework with a Hybrid Slow Learner and LoRA on ResNet-50 to reduce catastrophic forgetting for satellite imagery and poverty mapping.

**Kubuku: Financial Management for UMKM**  
Built multimodal extraction with Gemini 2.0 Flash over text, ASR voice, and OCR image inputs. Added hybrid dense/BM25 Q&A and a LangGraph ReAct agent for credit risk, inventory valuation, and cashflow analytics.
